# mitsuru-persona — 「第二の甲斐 充」

このリポジトリは、甲斐 充さんの分身AI「第二の甲斐 充」の人格データを保存する**privateリポジトリ**です。

## 目的

- `persona/mitsuru.md` — 人格文書（分身の記憶・人格のすべて）
- `interviews/` — インタビューQ&Aの生ログ（日付ごと）

## 使い方（2つのスキル）

- `/interview` — Claudeがインタビュアーになり質問を重ね、回答をペルソナ文書に蒸留して分身を育てる
- `/kabeuchi` — Claudeが「甲斐 充」本人として一人称で応答する壁打ちモード

## 絶対のルール

- **人格データ（persona/ と interviews/ の内容）をこのリポジトリの外に出さないこと。**
  公開リポジトリへの転記、外部サービスへの送信、他リポジトリへのコピーは禁止
- このリポジトリがprivateであることが前提。公開化の操作は行わない
- 変更（インタビュー結果・ペルソナ更新）は必ずコミットして `git push -u origin main` で永続化する
  （リモートセッションのコンテナは使い捨てのため、プッシュしないと消える）
