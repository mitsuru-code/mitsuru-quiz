# 📱 PC不要・Pixel 10a だけで回す X収益化運用ガイド

このリポジトリの「時事クイズ Bot 管理」アプリは、**スマホ1台で完結する**ように設計されています。
このガイドは、Pixel 10a を最大限活用して X（旧Twitter）の収益化を目指すための運用手順書です。

---

## 1. X収益化の条件（クリエイター広告収益分配）

アプリの「💰 X収益化トラッカー」はこの3条件の進捗を管理します。

| 条件 | 目標値 | 確認場所（Xアプリ内） |
|---|---|---|
| X Premium 加入 | 加入済み | プロフィール → プレミアム |
| フォロワー数 | **500人以上** | 自分のプロフィール |
| インプレッション | **直近3ヶ月で500万以上** | プレミアム → アナリティクス |

> ⚠️ 条件はXの規約変更で変わることがあります。最新の条件は Xアプリ →「収益化」画面で必ず確認してください。

### トラッカーの使い方
1. 週1回、Xアプリのアナリティクスでフォロワー数とインプレッションを確認
2. クイズBotアプリの「💰 X収益化トラッカー」に数値を入力 →「💾 数値を更新」
3. 進捗バーが100%（緑色）になった項目から順に条件クリア

投稿ペース（今日の投稿・直近7日・連続投稿日数）は、アプリ内で承認したクイズから**自動集計**されます。

### 推定月間収益の試算（非公式の目安）

トラッカーには「💴 推定月間収益」という試算欄もあります。以下の2つを入力すると、月間の目安収益（円）を自動計算します。

| 入力項目 | 確認場所（Xアプリ内） |
|---|---|
| 直近28日のインプレッション | プレミアム → アナリティクス（「28日間」表示になっている数値） |
| 想定単価（円 / 1万インプレッション） | 自分で入力・調整する仮の数字 |

> ⚠️ **Xはクリエイター広告収益分配の単価を公表していません。** この試算は完全に非公式の目安であり、実際の入金額を保証するものではありません。単価欄には初期値として仮の数字が入っていますが、実際に収益化が始まって入金額が分かったら、`入金額 ÷ (インプレッション数 ÷ 10000)` で自分の実績単価を逆算し、単価欄をその数字に更新してください。使うほど精度が上がる仕組みです。

なぜ「3ヶ月合計」ではなく「直近28日」を別途入力するのか: 3ヶ月合計を3で割ると、フォロワーが急増中/急減中のときに大きく実態とズレます。Xアナリティクスにはそのまま「直近28日」の数値が表示されるので、それを直接使うほうが月間ペースの実態に近い試算になります。

## 2. Pixel 10a の初期セットアップ

### PWAとしてインストール（必須）
1. Chrome でこのアプリのURLを開く
2. メニュー（⋮）→「ホーム画面に追加」→「インストール」
3. ホーム画面の「クイズBot」アイコンから起動（全画面・アプリとして動作）

### 通知を許可
- 初回起動時の通知許可ダイアログで「許可」を選択
- 定時リマインダー（7時・12時・21時）と速報検知の通知が届くようになります

### 画面スリープ対策（重要）
このアプリは**画面を開いている間のみ**動作します。運用時間帯は以下を設定:

- 設定 → ディスプレイ →「画面消灯」を **30分** に延長
- 充電中に運用する場合: 開発者向けオプション →「充電中は画面をスリープにしない」をON
- Chrome の電池最適化を解除: 設定 → アプリ → Chrome → バッテリー →「制限なし」

### 分割画面で「生成 → 投稿」を高速化
1. クイズBotアプリを開いた状態で、最近使ったアプリ画面を開く
2. アプリアイコンをタップ →「分割画面」→ Xアプリを選択
3. 上：クイズBot / 下：X の2画面で、コピー → 貼り付け → 投稿が数秒で完結

## 3. 1日の運用ルーティン（所要時間: 合計15〜20分）

| 時刻 | やること | 使う機能 |
|---|---|---|
| 🌅 7:00 | モーニングクイズを承認して投稿 | 定時リマインダー |
| ☀️ 12:00 | ランチクイズを承認して投稿 | 定時リマインダー |
| 随時 | 速報が来たら即クイズ化して投稿 | 🔴 速報ニュース監視 |
| 投稿の2時間後 | 正解を返信（通知が来る） | 💬 正解返信待ち |
| 🌙 21:00 | ナイトクイズを承認して投稿 | 定時リマインダー |

### 週次・月次
- **毎週日曜**: 「📅 週間まとめ」を生成 → X記事（長文）として投稿。滞在時間が伸び、インプレッション単価が上がります
- **毎月末**: 「📚 月間まとめ」を生成 → ストック資産として投稿
- **週1回**: 収益化トラッカーの数値を更新して進捗確認

## 4. インプレッションを最大化するコツ

1. **1日3投稿を絶やさない** — トラッカーの「連続投稿日数」を伸ばすことが最優先。アルゴリズムは継続アカウントを優遇します
2. **正解返信は必ず行う** — 返信でスレッドが伸びると、元ポストのインプレッションも再燃します
3. **𝕏引用ポストモードを活用** — 話題のポストを引用したクイズは、元ポストの流入を取り込めます
4. **投稿時間を守る** — 7時・12時・21時は日本のXのゴールデンタイム。リマインダーに従うだけでOK
5. **速報は速さが命** — 速報監視ONにしておき、通知が来たら5分以内に投稿
6. **回答リプライには「いいね」で反応** — 参加者が増えるほどエンゲージメント率が上がります

---

## 5. 運用上の注意

- **承認制を守る** — このアプリは全投稿を人間が承認・手動投稿する設計です。Xの自動化ルールに抵触しないための仕組みなので、無承認の自動投稿に改造しないこと
- **APIキーは端末内のみ** — Anthropic APIキーはこの端末のブラウザ内にだけ保存されます。端末を手放す際は「🗑 削除」を実行
- **同じ問題の再投稿は間隔を空ける** — 過去問題ライブラリから再利用する場合は1ヶ月以上空けるとスパム判定を避けられます

## 6. PCの自動化システムと連携する

PCで別の自動化システム（n8n / Make / 自作スクリプト等）を動かしている場合、アプリの「🔗 PC連携」カードから接続できます。承認したクイズをリアルタイムでPC側へ送信（Webhook）したり、全履歴のJSONエクスポート/インポートでデータを共有できます。

セットアップ手順・ペイロード仕様・PC側サンプルコードは **[pc-integration.md](./pc-integration.md)** を参照してください。
