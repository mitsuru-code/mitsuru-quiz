# ⚙️ Android/Pixel端末側の設定ガイド

このガイドは **Android端末本体の設定** に特化した文書です。アプリ内の⚙️設定カード（生成間隔など）の操作は [smartphone-operation-guide.md](./smartphone-operation-guide.md)、日々の運用の流れは [pixel10a-x-monetization-guide.md](./pixel10a-x-monetization-guide.md) を参照してください。

> ⚠️ 以下のメニュー名・階層はAndroidのバージョンによって多少表記が異なる場合があります。Pixelはほぼ素のAndroidのため大きくは変わりませんが、見当たらない場合は「設定」アプリの検索窓に太字のキーワードを入力して探してください。

---

## 1. PWA（ホーム画面アプリ）としてインストールする

1. ChromeでこのアプリのURLを開く
2. 右上のメニュー（⋮）→「ホーム画面に追加」または「アプリをインストール」をタップ
3. 「インストール」を選択
4. ホーム画面に追加された「クイズBot」アイコンから起動する

**確認方法**: ホーム画面のアイコンから起動したとき、Chromeのアドレスバーやタブが表示されず全画面表示になっていればPWAとして正しく動作しています。ブラウザのタブとして開いている場合はインストールがうまくいっていないので、手順1〜3をやり直してください。

---

## 2. 通知を確実に受け取る設定

定時リマインダー（7時・12時・21時）や速報検知の通知を受け取るために必要です。

1. 初回起動時に表示される通知許可ダイアログで「許可」を選択する
2. 誤って「ブロック」した場合の後からの設定: 設定 → アプリ → Chrome → 通知 → ON
   （PWAとしてインストール済みの場合は、設定 → アプリ →「クイズBot」→ 通知 → ON という項目が別途表示されることもあります）
3. 「おやすみモード」「サイレントモード」「集中モード」がONになっていると、通知が届いても音や表示が抑制されます。運用時間帯はOFFにするか、通知の例外に登録してください

**通知が来ないときのチェックリスト**:
- [ ] 通知の許可がONになっているか（手順2）
- [ ] おやすみモード・サイレントモードがOFFか
- [ ] アプリ（タブ）が開いたままになっているか（このアプリは画面を開いている間のみ動作します）
- [ ] 電池最適化でChromeの動作が制限されていないか（→ 5章）

---

## 3. 画面消灯（スリープ）を延長する

このアプリは**画面を開いている間のみ**動作する設計のため、画面がすぐ消えると自動生成・リマインダー・速報監視が止まります。

1. 設定 → ディスプレイ → 画面消灯（スリープ）
2. 運用時間帯は **30分** 程度に延長する
3. 短時間で確実に見るだけなら初期設定のままでも構いませんが、「ながら運用」する場合は延長を推奨します

---

## 4. 開発者向けオプションで「充電中はスリープにしない」を設定する

充電しながら長時間運用する場合におすすめです。

1. 設定 → 端末情報 → ビルド番号を連続で7回タップ（「これでデベロッパーになりました」等のメッセージが出る）
2. 設定 → システム → 開発者向けオプション を開く
3. 「スリープモードにしない（充電中は画面をオフにしない）」をON

> 開発者向けオプションは端末の動作に影響する項目も含まれるため、この設定以外は不用意に変更しないことをおすすめします。

---

## 5. Chromeの電池最適化を解除する

Androidの「適応バッテリー」機能により、バックグラウンド気味とみなされたアプリの動作（通知・タイマー）が制限されることがあります。

1. 設定 → アプリ → Chrome（またはインストール済みの「クイズBot」）→ バッテリー
2. 「制限なし」を選択（初期値は「最適化」または「制限あり」になっていることが多い）

これにより、定時リマインダーや速報監視のタイマーがOS側の省電力機能で止められにくくなります。

---

## 6. 分割画面でクイズBotとXを同時に表示する

クイズの承認・投稿はクイズBotとXアプリを行き来する操作が多いため、分割画面にすると効率化できます。起動方法のみ以下に示します。具体的なタップ手順（承認〜投稿の流れ）は [smartphone-operation-guide.md](./smartphone-operation-guide.md) の「11. 分割画面でクイズBotとXを同時に開く」を参照してください。

1. クイズBotアプリを開いた状態で、最近使ったアプリ一覧（画面下から上にスワイプして止める、または四角形ボタン）を開く
2. アプリのアイコンをタップ →「分割画面」を選択
3. 下側の画面に表示するアプリとしてXを選択する

---

## 7. ストレージ・データ管理の確認

このアプリのクイズ履歴・設定・APIキーは、すべて**ブラウザのlocalStorage**（この端末のChrome内）に保存されています。

- 設定 → アプリ → Chrome →（または個別サイトの）ストレージとキャッシュ →「データを削除」を実行すると、**保存されているクイズ履歴・設定・APIキーがすべて消えます**
- キャッシュのクリアとサイトデータの削除は別物です。「キャッシュを削除」だけならデータは残りますが、「サイトデータを削除」または「ストレージを消去」を選ぶと消えるので注意してください
- 消えると困る場合は、アプリの「⬇️ エクスポート」で定期的にJSONへ書き出しておくことを推奨します（詳細: [README.md 4章](../README.md#4-データの保存とプライバシー)、[pc-integration.md](./pc-integration.md)）

## 8. うまく動かないときの端末側チェックリスト

| 症状 | 疑うべき端末設定 |
|---|---|
| 通知が来ない | 2章（通知許可・おやすみモード）、5章（電池最適化） |
| 画面を離れるとすぐ生成・監視が止まる | 3章（画面消灯時間）、4章（充電中スリープしない） |
| 長時間運用すると動作が不安定になる | 5章（電池最適化を「制限なし」に） |
| 更新したはずのデータが元に戻る/消えた | 7章（サイトデータを誤って削除していないか） |
| ホーム画面から開くとブラウザのタブになってしまう | 1章（PWAとして正しくインストールされているか再確認） |
