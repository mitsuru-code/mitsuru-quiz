# 📱 TikTok Liteをバックグラウンドで再生するガイド

TikTok Liteを他のアプリを使いながら・画面を見ずに再生したい場合の方法をまとめた、Android/Pixel端末向けのガイドです。クイズBotの運用と並行して「ながら見」する使い方も紹介します。

> ⚠️ メニュー名・画面構成はTikTok Liteのバージョンによって変わることがあります。見当たらない場合はアプリ内の検索や設定画面を確認してください。

---

## 1. まず知っておくこと（重要な制約）

- **通常動画（おすすめフィード）には公式のバックグラウンド再生機能はありません。** ホームに戻ったり画面を消したりすると再生は止まります
- **LIVE配信のみ**、ピクチャーインピクチャー（PiP: 小窓表示）でのバックグラウンド視聴に対応しています（→ 2章）
- **ポイ活の注意**: TikTok Liteのポイント（動画視聴タスク）は、**バックグラウンド再生・PiP・画面オフの状態では付与されない仕様**です。ポイント目的の場合はアプリを前面に表示して視聴する必要があります
- 「画面オフでも再生できる」「自動でポイントが貯まる」とうたう**改造アプリ・非公式ツールは使わないでください**。アカウント停止やマルウェア感染のリスクがあります

---

## 2. LIVE配信をバックグラウンド（PiP）で再生する

LIVE配信は小窓（ピクチャーインピクチャー）にして、他のアプリを使いながら視聴できます。

1. まず端末側でPiPを許可する: 設定 → アプリ → TikTok Lite → **ピクチャーインピクチャー** → 「ピクチャーインピクチャーの使用を許可」をON
2. TikTok LiteでLIVE配信を開く
3. 画面内の**シェア（矢印）アイコン** → 「**バックグラウンドで再生**」（または「ピクチャーインピクチャー」）をタップ
   - この項目が見当たらない場合は、LIVE視聴中にそのままホームに戻る（画面下から上にスワイプ）と小窓化されることもあります
4. 小窓はドラッグで移動でき、タップで一時停止・全画面復帰などの操作ができます

> 💡 小窓の状態でも音声は流れ続けるため、「音だけ聞きたい」場合にも実質この方法が使えます（小窓を画面の隅に寄せておく）。

---

## 3. 通常動画を「ながら見」する — 分割画面

通常動画はバックグラウンド再生できないため、**分割画面**で画面の半分に表示し続けるのが現実的な方法です。クイズBotやXアプリと並行して使えます。

1. TikTok Liteを開いた状態で、最近使ったアプリ一覧（画面下から上にスワイプして止める）を開く
2. TikTok Liteのアイコンをタップ → 「**分割画面**」を選択
3. もう片方の画面に表示するアプリ（クイズBot・X・ブラウザなど）を選択する

- 分割画面ならTikTok Liteは「前面表示」の扱いなので、再生が止まりません
- 画面消灯すると止まるため、長時間使う場合は画面消灯時間の延長を推奨します（→ [pixel10a-device-settings-guide.md 3章](./pixel10a-device-settings-guide.md#3-画面消灯スリープを延長する)）

---

## 4. ブラウザ版で音声だけ流す（補助的な方法）

Chromeでブラウザ版TikTok（tiktok.com）を開いて動画を再生し、ホームに戻ると、**通知シェードのメディアコントロールから再生を再開**でき、音声だけ流し続けられる場合があります。

1. ChromeでTikTokのサイトを開き、動画を再生する
2. ホームに戻る（再生はいったん止まる）
3. 通知シェードを下ろし、メディア通知の「▶ 再生」をタップ

> ⚠️ この方法はChrome・サイト側の仕様変更で使えなくなることがあり、動作は保証されません。また1本の動画が終わると次へは進みません。連続再生したい場合は2章（LIVEのPiP）か3章（分割画面）を使ってください。

---

## 5. クイズBot運用と併用するときのおすすめ構成

このリポジトリのクイズBotも「画面を開いている間のみ動作する」設計のため、TikTok Liteと併用するなら次の構成が両立しやすいです。

| やりたいこと | おすすめの方法 |
|---|---|
| クイズBotを動かしながらTikTok Liteを見る | 分割画面（上: クイズBot / 下: TikTok Lite）（→ 3章） |
| LIVEを聞き流しながら他の作業をする | LIVEをPiP小窓にして隅に置く（→ 2章） |
| ポイントを貯めたい | バックグラウンドでは貯まらないため、TikTok Liteを前面（または分割画面）で視聴する |

- 分割画面での長時間運用は電池消費が大きいため、充電しながらの利用を推奨します（→ [pixel10a-device-settings-guide.md 4章](./pixel10a-device-settings-guide.md#4-開発者向けオプションで充電中はスリープにしないを設定する)）

---

## 6. うまくいかないときのチェックリスト

| 症状 | 確認すること |
|---|---|
| LIVEを小窓にできない | 端末設定でTikTok LiteのPiPが許可されているか（→ 2章手順1）。LIVE以外の通常動画はPiP非対応 |
| ホームに戻ると再生が止まる | 通常動画の仕様です。分割画面（→ 3章）を使う |
| バックグラウンドで見たのにポイントが増えない | 仕様です。ポイントは前面での視聴のみ対象（→ 1章） |
| 分割画面にすると再生が止まる・カクつく | 電池最適化でTikTok Liteが制限されていないか: 設定 → アプリ → TikTok Lite → バッテリー → 「制限なし」 |
